Food Market App

Create an app Food Market for a customer. 
It's a big company that deals mostly  with fruits and vegetables.

Main menu:
1. Create a new item in the stock.
2. Add item to the stock.
3. Sale the item.
4. Stock report.
5. Edit item in the stock.
6. Exit.


Try to create a website using Django. Online app for Food Market.
